# Digital Image Processing - 1st Project 
The first assignment for the Digital Image Processing Course. Basic filtering applications on grayscale images.
Some High-Pass, Low-Pass and Directional Filters were developed, using Gaussian, Butterworth and Ideal methods.

## Development
The project was developed on VS Code using Python 3.10.10. The following core libraries were utilized:
* NumPy 2.1.1
* Matplotlib 3.10.8

## Execution and Reproducibility
As long as the above library and software versions(or compatible ones) are used, anyone can  execute the 'demo.py', presenting some filtering results, as images and spectrum plots. 
* The demo file must be kept with the whole project in order to run correctly. Also, the image for processing must be added in the same directory manually.
* The image file for processing should be named 'cameraman.png'. Otherwise, change the source code in 'demo.py' accordingly.
* The demo script can be executed using a Python Compatible code editor/environment(VS Code, Pycharm etc.) or via the command line by typing the command: '``python demo.py``' (In systems with Python pre-installed).
